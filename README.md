Autonomous Bot

An autonomous robot built using Arduino that can navigate its surroundings and make movement decisions without human intervention using onboard sensors.

Overview

This project focuses on developing a self-navigating robot capable of detecting obstacles and adjusting its path automatically. The bot combines sensor inputs with control logic to enable autonomous movement in dynamic environments.

Features

Autonomous navigation without manual control
Real-time obstacle detection and avoidance
Continuous environment monitoring
Automatic decision-making for movement
Integrated motor control system

Hardware Used

Arduino Uno/Nano
Ultrasonic Sensor (HC-SR04)
Motor Driver Module (L298N or equivalent)
DC Motors
Robot Chassis
Wheels
Battery Pack
Connecting Wires

Software Used

Arduino IDE
C/C++ (Arduino)

Project Workflow

Read distance data from the ultrasonic sensor.
Detect nearby obstacles.
Process sensor inputs using control logic.
Decide the next movement direction.
Control motors for autonomous navigation.


Results

Successfully detected and avoided obstacles.
Achieved autonomous movement in indoor environments.
Validated sensor integration and motor control.


Future Improvements

Add Bluetooth/Wi-Fi remote monitoring.

Add path planning and mapping capabilities.
